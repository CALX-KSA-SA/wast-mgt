# CALX Tire Recycling Proposal Website

This is a high-end, static HTML/CSS/JS website designed for the CALX Engineering & Consulting Technical Proposal for Tires and Industrial Waste Recycling.

## Features
- **Design:** Modern "Glassmorphism" aesthetic with vibrant neon accents and dark mode.
- **Responsiveness:** Fully responsive layout for Desktop and Mobile devices.
- **Interactivity:**
  - Animated "Input -> Process -> Output" visualizations.
  - Smooth scroll navigation.
  - Interactive "Order / Quote Request" form.
  - WhatsApp integration for direct communication.
- **Content:** Fully localized Arabic content aligned with Saudi Vision 2030 standards.

## Project Structure
- `index.html`: Main entry point.
- `css/`: Stylesheets (`style.css`, `themes.css`, `animations.css`).
- `js/`: Application logic (`app.js`).
- `assets/`: Images and icons (e.g., Vision 2030 logo).

## How to Run / Deploy
This is a static website and can be deployed on any standard web hosting service (GitHub Pages, Netlify, Vercel, Apache, Nginx).

### Local Development
1. Clone the repository.
2. Open `index.html` in your browser.
   - OR run a simple local server: `python3 -m http.server 8000`

### GitHub Pages Deployment
1. Go to your repository Settings > Pages.
2. Select the `main` branch and `/ (root)` folder.
3. Save. Your site will be live at `https://<username>.github.io/<repo-name>/`.
